Settings
Todoist API token
Your Todoist API token can be found in Todoist Settings under the Integrations menu. This API token, as well as any Todoist data, is only stored locally on your computer and is never sent to Todayist.
Hour to start the day
Your hour to start the day is used to set the first task in case its duration is not specified. You can specify the estimated duration for its task by appending $minutes to its name on Todoist. For example, Do Pushups 18:00 $30 #Fitness is a task that will appear on the after-noon graph from 18:00 to 18:30. Todayist only picks the tasks that you have for Today, excluding the overdue and the ones without due time.
